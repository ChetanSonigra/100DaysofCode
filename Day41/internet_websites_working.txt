--> How do internet work?

some computers are servers which has to be 24x7 and serves the data you need.

client(browser) --asks data through url--> ISP(internet service provider) 
--checks ip address--> DNS (Domain Name Server)

checkout nslookup.io to find ip address for any site. you can use this ip to get the website.
