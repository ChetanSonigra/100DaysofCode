--> HTML Boilerplate:
to understand structure of html
type ! to generate boiler plate

<!DOCTYPE html>    -- to say that we have written this file in html5
<html lang="en">   -- root tag
  <head>           -- important info about website. 
    <meta charset="UTF-8">
    <title>Title to be displayed in tab</title>
  </head>
  <body>        -- all the contents of website.
    <h1>Hello World</h1>
  </body>

</html>
